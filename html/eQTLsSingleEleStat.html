eQTLsSingleEleStat {NetLCP}R Documentation

single element eQTLs statistics

Description

single element eQTLs statistics

Usage

eQTLsSingleEleStat(regData, eQTLsData, filterDegree, selectNode)

Arguments

regData

the standard output of the function binaryRegulationExtract or multieleRegulation.

eQTLsData

the standard out of the function eQTLsDetection.

filterDegree

an integer for filtering the nodes.

selectNode

select a or a set of nodes to perform statistics.

Value

single element eQTLs statistics with bar plot.

Examples

 eQTLsSingleEleStat(regData = NULL, eQTLsData = NULL, filterDegree = 100,selectNode = "MIMAT0000423",netLayout = "layout_in_circle")
