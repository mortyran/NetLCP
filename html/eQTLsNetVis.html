eQTLsNetVis {NetLCP}R Documentation

eQTLs Regulatory network visualization

Description

eQTLs Regulatory network visualization based on regData and eQTLsData.

Usage

eQTLsNetVis(regData, eQTLsData, filterDegree, selectNode, netLayout)

Arguments

regData

the standard output of the function binaryRegulationExtract or multieleRegulation.

eQTLsData

the standard out of the function eQTLsDetection.

filterDegree

an integer for filtering the nodes.

selectNode

select a or a set of nodes to visulize in the network.

netLayout

the layout of network.

Value

eQTLs Regulatory network visualization

Examples

 eQTLsNetVis(regData = NULL, eQTLsData = NULL, filterDegree = 100,selectNode = "MIMAT0000423",netLayout = "layout_in_circle")
